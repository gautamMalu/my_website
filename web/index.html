IOT What & Why

Putting it simply, IOT consists of devices (such as a sensor) which are connected to a machine in order to capture an event which in turn is relayed over a network delivering data to applications. These sensors are wirelessly connected to each other and the central server. They are able to receive and transmit the event data wirelessly via a central server where an Application translates this data into meaningful information. This information can be analysed further and can invoke appropiate response
